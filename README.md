# tekisei-ad (適正広告費計算ツール)

あなたのビジネスで「広告費をいくら使えばいいか」を直感的に知るためのWebアプリです。

## 概要
専門用語を一切排除し、初心者の方でも「1人集めるのにいくら使っていいのか」「目標達成には何人集める必要があるのか」を一目で理解できるように設計されています。

## 主な機能
- **3ステップ入力**: 画面の案内に沿って数値を入力するだけで計算完了。
- **適正広告費の自動判定**: 利益率や成約率をもとに、無理のない広告予算を算出。
- **コンサル機能（警告表示）**: 成約率が低すぎる場合や、集客が難しい場合にアドバイスを表示。
- **レスポンシブ対応**: スマートフォンからでも快適に入力・閲覧が可能。

## 計算項目
1. **1人あたり広告費 (限界CPA)**: 1人のお客さんを集めるのに使える最大金額。
2. **必要な集客数**: 目標売上を達成するためにサイトや店舗に呼ぶべき人数。
3. **月の適正広告費**: ビジネスモデルに基づいた、ちょうどいい広告予算。

## 使用技術
- **Next.js (App Router)**
- **TypeScript**
- **Vanilla CSS (CSS Modules / Globals)**
- **Framer Motion** (アニメーション)
- **Lucide React** (アイコン)

## インストールと実行
1. 依存関係のインストール:
   ```bash
   npm install
   ```
2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```
3. ブラウザで `http://localhost:3000` を開く。
